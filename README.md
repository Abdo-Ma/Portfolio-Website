## ✨ Features

- **Modern Responsive Design**  
  Looks great on all devices (desktop, tablet, mobile).

- **Projects Showcase**  
  Interactive cards for each project with images, tech stack, and links.

- **Search & Filter**  
  Instantly search and filter projects.

- **Show More/Less**  
  Expand to view more projects dynamically.

- **Dark/Light Mode**  
  Toggle between dark and light themes.

- **Navigation Bar**  
  Smooth navigation with scroll and mobile menu support.

- **Back to Top Button**  
  Quick scroll to the top of the page.

- **Loading Animation**  
  Engaging loader for better UX.

- **404 Error Page**  
  Custom error page for broken links.

- **Testimonials & Team**  
  Dedicated sections for client feedback and team members.

## 📑 Getting Started

1. **Clone the repository**

   ```sh
   git clone https://github.com/your-username/your-portfolio.git
   cd your-portfolio
   ```

2. **Open `index.html` in your browser**  
   No build tools required—just open and go!

---

## 📋 Customization

- **Add/Edit Projects:**  
  Update the `source/Javascript/script.js` or `main.js` with your project details and images in `assets/Images/Projects/`.

- **Change Profile Info:**  
  Edit `index.html` and relevant sections in `source/HTML/`.

- **Update Images:**  
  Replace images in `assets/Images/` as needed.
